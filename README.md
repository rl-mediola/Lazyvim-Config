# Shortcuts
## Normal mode

| Key    | Action            |
| :----- | :---------------- |
| m      | write buffer      |
| M      | reset buffer      |
| u      | undo              |
| \<C-u> | redo              |
| \<C-q> | close             |
| .      | toggle upper case |

## Insert mode

| Key    | Action                        |
| :----- | :---------------------------- |
| ;;     | enter normal mode             |
| \<C-l> | delete character to the right |
| \<C-h> | delete character to the left  |
