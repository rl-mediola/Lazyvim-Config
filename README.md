# 💤 LazyVim

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

# Prerequisites

apt install unzip ripgrep fd

# Shortcuts

## Shortcuts from VIM and lazyvim

| Key        | Action                  |
| :--------- | :---------------------- |
| \<CTRL>O   | jump back               |
| \<CTRL>I   | jump forth              |
| \<SPACE>ch | switch to header/source |

## Normal mode

| Key             | Action       |
| :-------------- | :----------- |
| \<ALT>[h,j,k,l] | fast scroll  |
| m               | write buffer |
| M               | reset buffer |
| u               | undo         |
| \<CTRL>u        | redo         |
| \<CTRL>q        | close        |

## Insert mode

| Key             | Action                        |
| :-------------- | :---------------------------- |
| \<ALT>[h,j,k,l] | cursor movement               |
| ;;              | enter normal mode             |
| \<CTRL>l        | delete character to the right |
| \<CTRL>h        | delete character to the left  |
